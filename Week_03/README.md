# 字符串分析算法
1. 字典树: 大量搞重复字符串的存储与分析 (应用： 处理大量的输入和字符串)
  字典树是hash树的一个特例，hash树在字符串领域里，最直接的应用体现就是字典树
2. KMP: 在长字符串里找模式，复杂度o(m+n), leetcode对应26题
  KMP是字符串的模式匹配算法（查一个字符串里有没有另一个字符串），原串&pattern串组成
3. Wildcard: 带通配符的字符串模式
4. 正则: 字符串通用模式匹配
5. 状态机: 通用的字符串分析
6. LL LR 字符串多层级结构分析
